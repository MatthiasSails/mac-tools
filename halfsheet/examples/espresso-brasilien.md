# Espresso — Brasilien Arabica Natural

COOPFAM · 100 % Arabica Natural · Ernte 2025
**Aromen:** Karamell · Malz · Getrocknete Feige

---

## Rezept (Cappuccino)

| Parameter       | Wert      |
| --------------- | --------- |
| Kaffeemenge     | 18.5 g    |
| Espresso Output | 42 g      |
| Ratio           | ca. 1:2.3 |
| Bezugszeit      | 30 s      |
| Temperatur      | 93.5 °C   |

---

## Mahlgrad

mittel-fein · eher gröber als helle Röstungen · nicht „turbo-fein"

---

## Workflow

1. Mahlen
2. WDT
3. Leveln
4. Tampen
5. Bezug starten

---

## Korrekturen

| Problem             | Korrektur          |
| ------------------- | ------------------ |
| bitter / trocken    | gröber mahlen      |
| sauer / dünn        | feiner oder heißer |
| flach / schwer      | längerer Shot      |
| zu schwach in Milch | 19–20 g Dose       |

---

## Zielgeschmack

**Süß · karamellig · vollmundig · wenig Säure · ideal für Cappuccino**
